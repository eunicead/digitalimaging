This program swaps faces from two images in "result.jpg"
There are some images and results in two folders.

You have to downoad and unzip http://sourceforge.net/projects/dclib/files/dlib/v18.10/shape_predictor_68_face_landmarks.dat.bz2 
This file has to be in the same directory than projet.py
To launch the program :
python projet.py image1 image2
